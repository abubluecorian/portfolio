Question 6

How many times will the following code print the word 'Hello'?

for(var i = 0; i <= 5; i++) {
    console.log("Hello");
}

1 / 1 point

Correct! 'i' starts with the value '0'. The condition checks if 'i' is less than or equal to '5'. Each loop increments 'i' by '1'. This means that the loop will run 6 times.
