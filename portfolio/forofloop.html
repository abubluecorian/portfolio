Task: Exploring Array Iteration

In this exercise, we'll practice iterating over an array using the for...of loop.

Step 1

You have a list of fruits:

            var fruits = ['apple', 'banana', 'orange', 'strawberry', 'kiwi', 'grapes']

Create a function named listFruits. Inside this function, use the for...of loop to display each item in the fruits array.

Once you've defined the function, call it as listFruits() to see the list of fruits displayed in the console.

The expected output should be:

            apple
            banana
            orange
            strawberry
            kiwi
            grapes

Step 2

Here's an array of countries:

            var countries = ['USA', 'Canada', 'Australia', 'Japan', 'France', 'Brazil']

Define a function called displayCountries. Within this function, use the for...of loop to show each country from the countries array.

Execute the function by calling displayCountries() to view the list of countries in the console.

Expected output:

            USA
            Canada
            Australia
            Japan
            France
            Brazil
