JavaScript Code Explanation

 
        var h1 = document.createElement('h1');
        h1.innerText = "Type into the inputs to perform calculations";
        
        var input1 = document.createElement('input');
        input1.setAttribute('type', 'text');
        
        var input2 = document.createElement('input');
        input2.setAttribute('type', 'text');
        
        var selectOperation = document.createElement('select');
        var options = ["Addition (+)", "Subtraction (-)", "Multiplication (×)", "Division (÷)"];
        
        options.forEach(function(optionText, index) {
            var option = document.createElement('option');
            option.value = index;
            option.text = optionText;
            selectOperation.appendChild(option);
        });
        
        var resultInput = document.createElement('input');
        resultInput.setAttribute('type', 'text');
        resultInput.setAttribute('readonly', true);
        
        var submitButton = document.createElement('button');
        submitButton.innerText = "Calculate";
        
        // document.body.innerText = '';
        document.body.appendChild(h1);
        document.body.appendChild(input1);
        document.body.appendChild(selectOperation);
        document.body.appendChild(input2);
        document.body.appendChild(submitButton);
        document.body.appendChild(resultInput);
        
        submitButton.addEventListener('click', function() {
            var operation = parseInt(selectOperation.value);
            var result;
            
            switch (operation) {
                case 0:
                    result = parseInt(input1.value) + parseInt(input2.value);
                    break;
                case 1:
                    result = parseInt(input1.value) - parseInt(input2.value);
                    break;
                case 2:
                    result = parseInt(input1.value) * parseInt(input2.value);
                    break;
                case 3:
                    result = parseInt(input1.value) / parseInt(input2.value);
                    break;
                default:
                    result = "Invalid Operation";
            }
            
            resultInput.value = result;
        });
Calculator Explanation

Understanding JavaScript Code for Calculator

Creating HTML Elements:

h1, input1, input2, selectOperation, resultInput, and submitButton are created using document.createElement() for respective HTML elements.

Setting Attributes and Values:

h1.innerText sets the text content for the h1 element.

input1 and input2 are text input fields.

selectOperation is a dropdown menu (<select>) for operations.

options contains an array of strings for operation options.

forEach loop creates option elements for each operation in options and appends them to the selectOperation.

Appending Elements to the Document Body:

The elements created above (h1, input1, selectOperation, input2, submitButton, and resultInput) are appended to the document body using document.body.appendChild().

Button Click Event Listener:

submitButton.addEventListener listens for a click event on the submit button.

When clicked, it gets the selected operation from the dropdown.

Performs the corresponding operation based on the selected value (0 for addition, 1 for subtraction, 2 for multiplication, and 3 for division).

The result of the calculation is displayed in the resultInput field.

To convert this JavaScript code into an HTML file, you can place this script inside a <script> tag in the <body> section of an HTML file. This will create a simple calculator interface with input fields, a dropdown for operations, a button to calculate, and an output field to display the result.
